## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Gift-Sharing-NFT-WEB3.0
```

Go to the project directory

```bash
  cd Gift-Sharing-NFT-WEB3.0
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```
